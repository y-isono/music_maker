# test関数

## test_midi.pyについて
### 実行方法
```
python test_midi.py <midi file path>
```
#### requirements
- pretty-midi
- synthesizer

##### pretty-midiのインストール方法
```
git clone https://github.com/craffel/pretty-midi
cd pretty-midi
python setup.py install
```

##### synthesizerのインストール方法
```
pip install synthesizer
```
